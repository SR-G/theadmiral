The Admiral Versions

Version Notes - 1.4.0 (SRG)

  • Project stated again the sources of the 1.2 version available on sourceforge, with retro-inclusion of every modification done by SuperHomer in the meanwhile.
  • Project published on GitHub https://github.com/SR-G/theadmiral
  • Minor internal refactoring
  • Switch to Maven as a building tool
  • Card numbers from PotSM renamed
  • Added image preview of every card (provided that the corresponding .png file is available unded "img/expansions/", in a subfolder named by the expansion abbreviation, and with a picture name like "001-1.png" (example : "img/expansions/PofDJC/001-1.png"))

Version Notes - 1.3.9 (SuperHomer)

  • Added support for "Equipment" card type which is new to the Fire & Steel set.
  • Added "Navigator" & "Silver Explorer" buttons under Generic Crew on the Fleet tab.
  • Alphabetized Generic Crew buttons on Fleet tab.
  • Incorporated Pirates of the Cursed Seas: Savage Shores data.  (entered directly from cards by Homer)
  • Incorporated Pirates of the Cursed Seas: Fire & Steel data. (thanks to gleritte for the raw data)
  • Incorporated all corrections made to the database since prior release.

Version Notes - 1.3.7 (SuperHomer)

  • Added support for movement of "T" which is new to the Rise of the Fiends set.
  • Incorporated Pirates of the Caribbean & Rise of the Fiends data. (thanks to gleritte for the raw data)
  • Incorporated all corrections made to the database since prior release.

Version Notes - 1.3.5 (SuperHomer)

  • Added support for movement of "D" which is new to the Ocean's Edge set.
  • Added support for rarity code of "M" for "Mega Pack" which is new to the Ocean's Edge set.
  • Corrected name of new set from "Pirates of the Ocean's Edge" to "Pirates at Ocean's Edge".
  • Edited rarity sorter to place all Green coded card together.   Former version split P and V rarities with L rarity.
  • Incorporated all corrections made to the database since prior release.
  • Incorporated Ocean's Edge card data.  (thanks to gleritte for the raw data)
  • NOTE: Special Edition "SCS" cards only available in Ocean's Edge tins have a rarity code of "Value Tin (Special Edition)"

Version Notes - 1.3.2 (SuperHomer)

  • Added four specialists to the 'Add Generic Crew' panel.
  • Revised "Print Fleet" & "Export Fleet" functions to now include the set and card # for each card in fleet.
  • Corrected Rarity Sort so that Super Rares, Limited Editions, Promos and Treasures are sorted too.  (version 1.2 only sorted Commons, Uncommons & Rares)
  • Changed color of 'Limited Edition' rarity from Pink to Orange to more closely reflect the actual color scheme of the game.
  • Added support for green 'promo' rarity for ship's like "Santa's Sleigh" and found code already in place to support the green rarity cards that are only available in  value boxes like the "Independence" from a Mysterious Islands Value Box.    None of these are available to view on WizKids' web site.    If you wish to import them, make sure that you use "Value Box" or "Promo" for the rarity.
  • Updated database file to contain cards from all sets up to Frozen North, including Prize Cards, Value Box Ships and Con-Exclusives. (thanks to gleritte for updated database)
  • Modified Mercenary Flag to resemble actual flag in game. (thanks to riz for graphic)
  • Added flag & faction support for Vikings. (thanks to riz for graphic)
  • Added flag & faction support for North Pole. (thanks to gleritte for graphic)
  • Added support for Frozen North and Ocean's Edge sets. [Use "Pirates of the Ocean's Edge" to import]
  • NOTE: In the process of editing source code, found the ability to delete a single card from the database.   The addition of this function had been requested by a user but was not needed because it was already incorporated in original version.  (To delete a single card, click on the card's line anywhere except the 'Want' or 'Have' column to highlight it and press the 'Delete' key)
  • Updated copyright date and version number in the 'About' window.
  • Appended help file -this text- to include version updates.
